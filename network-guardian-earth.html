Workflow

Use the free Google Earth path first

Google Earth Standard supports cloud import and storage with a 1 GB limit, which is far above this small export.

Fast local view

Open Google Earth, choose New, then open a local KML file. Select the KML from this page to inspect the layer without publishing it.

Collaborative project

Sign in to Google Earth, create a project, and import the KML or GeoJSON. The project can be saved and shared from Google Drive.

Public website

For a fully embedded public map, use this same GeoJSON in a Google Maps or Leaflet page and keep raw scan fields out of the browser.
